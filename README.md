# Star Wars Example App
<br>
  
## Retrieving Starships By Pilot Name
<br>

### Via CLI
```
php artisan star-wars:get-starships-by-pilot-name "Luke Skywalker"
```

### Via API
```
GET /api/Starship/getByPilotName/Luke%20Skywalker
```
